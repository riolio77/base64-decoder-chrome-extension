# base64-decoder-chrome-extension
base64 decoder of chrome extension, just drag

---
# 사용법
1. 크롬 브라우저 접속
2. **주소창에 "chrome://extensions/" 접속** 혹은 **오른쪽 상단 메뉴 클릭 -> 확장 프로그램 -> 확장 프로그램 관리**
3. base64-decoder-extension 폴더 자체를 업로드(드래그 앤 드랍)
4. 크롬 재접속 혹은 새로고침
5. base64 로 인코딩된 텍스트 드래그

<br>

# How to Use
1. Open the Chrome browser
2. Go to "chrome://extensions/" in the address bar or click the top-right menu → Extensions → Manage Extensions
3. Upload the base64-decoder-extension folder itself (drag and drop)
4. Restart Chrome or refresh the page
5. Just Drag
